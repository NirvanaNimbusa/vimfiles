# My Vimfiles

Use at your own risk ;).

## Installing

    // Grab the files
    git clone https://github.com/carlosantoniodasilva/vimfiles.git ~/.vim

    // Download plugins with git submodules
    cd ~/.vim
    git submodule update --init

    // Link config files
    ln -s ~/.vim/vimrc ~/.vimrc
    ln -s ~/.vim/gvimrc ~/.gvimrc
